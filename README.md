# CoinKit

A minimal Discord bot for monitoring various crypto signals (DeFi and Governance).

<p align="center">
  <img src="assets/pfp.png" width="100">
</p>

The bot is designed to:
- track multiple types of on-chain metrics and discrete events (rates, caps, ICOs, etc)
- persist state across runs
- deliver alerts into Discord channels
- create GitHub issues on request
