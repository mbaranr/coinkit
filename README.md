# CoinKit

A minimal Discord bot for monitoring **DeFi protocol** metrics.

<p align="center">
  <img src="assets/pfp.png" width="100">
</p>

## What It Does

The bot is designed to:
- track multiple types of on-chain metrics (rates, caps, and more)
- persist state across runs
- deliver alerts into Discord channels
- create GitHub issues on request

## Protocols of Interest

- [Euler](https://www.euler.finance/)
- [Silo](https://www.silo.finance/)
- [Aave](https://aave.com/)
